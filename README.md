# Smart Envo

Smart Environment Firewall / Smart Firewall project built on Raspberry Pi.

## Project Structure
- `smartfirewall-backend/` - Flask backend, policy enforcement, schedules, monitoring, alerts, backups
- `smartfirewall-frontend/` - dashboard frontend for administration and monitoring

## Main Features
- Device management
- Group-based restrictions
- Domain blocking
- Service restrictions
- Scheduler
- Monitor / alerts / activity logs
- Backup and settings

## Notes
- Secrets and local deployment files are excluded from this repository.
- Use the provided `.env.example` files as templates.
- This repository reflects the final academic project submission state.
